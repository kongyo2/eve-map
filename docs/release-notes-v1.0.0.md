# リリースノート v1.0.0

## 初回リリース

EVE Onlineの宇宙「ニューエデン」を探索できるインタラクティブマップアプリの初回リリースです。

### 主な機能

- **宇宙マップ**: K-Space全域5,000以上のソーラーシステムとスターゲート接続をGPU加速で描画。ピンチ操作で拡大・縮小、ドラッグでスムーズに移動。ズームレベルに応じてリージョン/コンステレーション/システム表示を自動切替
- **システム情報**: タップでセキュリティステータス、所属リージョン・コンステレーション、ステーション一覧、スターゲート接続先を表示
- **ルート検索**: 最短・安全・危険の3種類のルート計算。回避システム設定、最寄りトレードハブへのルート表示に対応
- **リアルタイム統計**: ESI APIから直近1時間のキル統計・ジャンプ統計を取得。ヒートマップモードで危険宙域を可視化
- **キルメール・戦闘履歴**: 各システムの最近のキルメールと戦闘データを閲覧
- **主権情報**: ヌルセクのアライアンス主権をマップ上で確認
- **周辺検索**: 指定ジャンプ数以内のシステム検索。ステーション有無でのフィルタリング対応
- **ランドマーク検索**: ニューエデンの主要ランドマークを検索
- **マーケットデータ**: リージョンごとのマーケット価格情報を確認
- **全UI日本語対応**: システム名・リージョン名含む全テキストが日本語表示
- **オフライン対応**: 初回起動時にデータをキャッシュし、2回目以降は高速起動
- **広告なし・ログイン不要・個人情報収集なし**
